# Employee Management System (Java + JDBC)

## Description
A console-based Employee Management System built using Java, JDBC, and MySQL.
The application supports full CRUD operations with a normalized database design
using foreign keys and JOIN queries.

## Features
- Add new employee
- View all employees (with department & role)
- Update employee salary
- Delete employee
- Menu-driven console interface
- SQL Injection prevention using PreparedStatement

## Technologies Used
- Java
- JDBC
- MySQL
- IntelliJ IDEA

## Database Design
- employee
- department
- role
  (Relational design using foreign keys)

## How to Run
1. Configure MySQL credentials in DBUtil.java
2. Run MainApp.java
3. Use menu options
